Spring Boot Project

1. Prerequisites
   1. Initially you will need a JDK and Maven
      1. Use this link to set up JDK and Maven `https://www.youtube.com/watch?v=km3tLti4TCM`
   2. You will also need an IDE "Intellij"
   
   3. set up spring boot project at `https://start.spring.io/`

   4. download postman software to run requests locally `https://www.postman.com/`


2. Build and Run
   1. After you successfully installed JDK and MAVEN you will be able to build and run your code
   2. After code is running in terminal, head over to postman and use the get function
      1. use `http://localhost:8080/course` to run the course api locally
